# Security Policy

## Supported Versions

`e` is in early development. Security fixes are applied to the latest release on
the `main` branch.

| Version | Supported |
| ------- | --------- |
| 0.1.x   | ✅        |

## Reporting a Vulnerability

Please **do not** open a public issue for security vulnerabilities.

Instead, report them privately:

- Email **kh@gets.no**, or
- Use GitHub's [private vulnerability reporting](https://github.com/kwhorne/e/security/advisories/new).

Include a description of the issue, steps to reproduce, and the potential
impact. We aim to acknowledge reports within a few days and will keep you
informed of the progress toward a fix.

Thank you for helping keep `e` and its users safe.
