## Find & replace

| Shortcut | Action |
| -------- | ------ |
| `‚åòF`     | Find in file |
| `‚å•‚åòF`    | Replace in file |
| `‚åò‚áßF`    | Search in files (workspace) |
| `‚Üë` / `‚Üì` | Previous / next match (in the find bar) |

The find bar has toggles for **case-sensitive** (`Aa`), **whole-word** (`W`) and
**regular expressions** (`.*`).

The Source Control panel (`‚åò2`) provides stage, unstage, discard, stage-all,
